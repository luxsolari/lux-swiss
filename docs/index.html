Two rules govern everything

01 — Duotone strict

Ink and cream are the two functional colors; blood red is the lone accent. No success green, no info blue. Win/loss, active, emphasis, error — all weight, size, spacing, contrast.

02 — Swiss-minimalist

Visible 1px borders, no shadows — elevation is a background step. Generous whitespace, square corners, uppercase monospace labels with wide letter-spacing.

Medium

A single paragraph reads the same whether it's a card's supporting copy or a page's lead — same measure, same line-height, no separate rule for "short body text."

Long-form

Restraint scales. A single line of UI copy and a full article both draw from the same weight axis and the same measure — nothing about going longer asks for a new typographic idea.

What changes is paragraph spacing, not the voice: each paragraph gets the same top margin as the body rhythm established elsewhere on this page, so a long-form block reads as more of the same system, not a different one.
